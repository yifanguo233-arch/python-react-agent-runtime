# 面试资料目录

这个目录用于集中存放本项目的面试准备文档。

## 文档说明

- `interview_demo_plan.md`
  - 明天面试时可直接使用的项目演示方案
- `interview.md`
  - 项目整体介绍、Agent 相关知识点和高频问答
- `interview_code_flow_questions.md`
  - 面试官要求你现场顺着代码讲执行流程时的专项题库

## 建议使用顺序

1. 先看 `interview_demo_plan.md`
2. 再看 `interview.md`
3. 最后用 `interview_code_flow_questions.md` 做代码走读训练
