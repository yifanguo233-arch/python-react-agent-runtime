# write-tests

## Description
为指定文件或函数编写单元测试

## Keywords
- 写测试
- 单元测试
- 测试用例
- write test
- unit test
- 测试代码

## Steps
1. 读取目标文件，了解其函数和类的结构
2. 为每个公开函数设计测试用例（正常路径 + 边界情况）
3. 将测试代码写入 test/ 目录下的对应测试文件
